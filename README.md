# Homebrew Tap for squrl

Homebrew formulae for [squrl](https://github.com/FloodCreux/squrl) -- a terminal-based HTTP and WebSocket client.

## Installation

```sh
brew tap FloodCreux/squrl
brew install squrl
```

Or install directly in one command:

```sh
brew install FloodCreux/squrl/squrl
```

## Upgrading

```sh
brew update
brew upgrade squrl
```

## Supported Platforms

| OS    | Architecture       |
|-------|--------------------|
| macOS | Apple Silicon (ARM) |
| macOS | Intel (x86_64)     |
| Linux | ARM 64-bit         |
| Linux | Intel (x86_64)     |
